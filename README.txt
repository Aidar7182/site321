Для работы с проектом, необходимо иметь БД Open Server.
Для запуска необходимо открыть файл test.php.
Нужно создать в БД таблицы: товар,пользователи,корзина,заказ.
У товара есть id, name,category,price,description(описание),image(путь к фотографии).
У пользователя есть id,login,password,email,num,status.
У корзины есть id,user_id,tovar_id.
